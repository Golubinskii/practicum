  Целью данного проекта является создание модели, которая будет прогнозировать на основе истроических данных о поведении клиентов
и расторжении договоров, уйдет ли клиент из банка в ближайшее время или же будет продолжать пользоваться услугами этого банка.

Были использованы  модели:
 - логистической регрессии,
 - обучающего дерева,
 - случайного леса.

Для борьбы с дисбаллансом данных использовались методы upsample и dowmsample.

Метрикой качества являестя F1-мера с допольнительным измерением AUC-ROC.


Максимально возможные значения F1=0.5974 и AUC-ROC=0.837 показала модель случайного леса с глобальным параметром n_estimators=77.
При этом использовался метод борьбы с дисбалансом upsample.
